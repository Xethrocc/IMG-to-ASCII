# Changelog

## 0.1.0.0 — 2026-03-01

### Added

- Initial release
- Single image to ASCII conversion (PNG, JPG, BMP, TIFF)
- JSON export with metadata (ID, name, tags, category, variants)
- Batch conversion mode for directories
- Terminal-aware resizing with aspect-ratio correction
- Configurable character ramp, dimensions, and metadata
- CLI with `convert` and `batch` subcommands via optparse-applicative
